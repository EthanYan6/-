## 使用

1. 修改 `config.py` 中的语雀 `token` 、`login` 和嘀嗒清单的 `cookies` 、`x-device` 

2. 然后执行程序：

```shell
python getStatisticalInfo.py
```

> 本程序使用python3.x版本，所有模块均为自带，不需要下载。
